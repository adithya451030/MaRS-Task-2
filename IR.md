This code uses an IR sensor to count objects in a very simple way. Actually this task is about to do with LCD but I don't know wheather my LCD is working or not. Even the basic printing also not shown in LCD. So I have shown in Serial Monitor.
In setup(), pin 13 is set as an input because the IR sensor gives its signal to that pin. Inside loop(), the program continuously reads the sensor value using digitalRead(IR) and stores it in val. When the sensor detects an object, its output becomes 0, so the condition if(val==0) becomes true. And then it starts counting the object.
By using external interrupts we can do more better way that if the obect is placed for more time and then removed it should count only one time. But because of LCD problem the time was not sufficient and I was not able to do within the time.
